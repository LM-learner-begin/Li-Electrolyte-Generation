# Dataset Overview (数据集说明)

本项目包含两个精心构建的材料学数据集。它们分别用于训练 **属性预测模型 (CGCNN)** 和 **三维晶体生成模型 (CDVAE)**。为了保证模型在真实的固态电池电解质领域中具备泛化能力，我们对原始数据进行了严格的物理约束和清洗。

---

# 1. 电化学窗口数据集 (ECW Dataset)
**用途：** 用于训练 CGCNN 图神经网络，使其具备精准预测无机材料形成能 ($E_f$) 或电化学窗口 (ECW) 的能力。

### 数据来源与构成
* **基础来源：** 数据来自 Materials Project (MP) 数据库中的密度泛函理论（DFT）计算数据。
* **数据规模：** 原始数据包含 16,001 种含锂化合物。
* **化学空间分布：** 涵盖了绝大多数已知体系，其中氧化物占 85.8%，硫化物占 2.6%，卤化物占 0.93% 等。平均电化学窗口为 0.79V。

### 数据清洗规则 (Data Cleaning)
1. **去重处理：** 保留具有唯一化学式的结构，防止数据冗余导致模型过拟合。
2. **结构致密性过滤：** 通过脚本过滤掉了通过 API 下载的非致密结构（如气相分子），确保生成的图谱能构建正常的邻接矩阵。清洗后最终可用高质量数据量为 **15,925** 条。

---

## 2. 锂离子固态电解质数据集 (Li-SSE Dataset)
**用途：** 用于训练 CDVAE 扩散模型，使 AI 学习真实固态电解质的三维空间分布规律，从而生成全新的、合理的候选材料。

### 数据来源与构成
* **基础来源：** 联合检索并筛选自 Materials Project (MP) 与无机晶体结构数据库 (ICSD)。
* **涵盖典型构型：** 数据集专门定向收集了具有良好离子电导率潜力的结构框架，包括：硫银锗矿与硫化物体系 (Li-S-P)、石榴石型氧化物 (如 LLZO)、NASICON 型磷酸盐 (如 LATP) 以及高压卤化物。
* **最终规模：** 经过极其严格的物理过滤后，最终用于训练的高质量标准晶体共计 **5,503** 个。对于无属性生成任务，使用了 Dummy Property 作为占位标签；对于条件生成任务，附带了 MP 数据库的形成能与带隙标签。

### 核心物理约束与筛选机制 (Strict Physical Filters)
为了防止 CDVAE 学习到无用的合金或不合理的假结构，我们在数据收集阶段引入了 4 项核心物理限制：
1. **纯无机含锂要求：** 必须包含 Li 元素，且 **严格剔除 H（氢）元素**，以排除有机物、聚合物或水合物的干扰。
2. **电子绝缘性 (Band Gap > 0.5 eV)：** 固态电解质必须传导离子但阻断电子。设定 0.5 eV 作为安全下限（考虑到 DFT 会低估带隙），精准剔除了会导致电池短路的合金和半导体。
3. **热力学高稳定性 ($E_{hull} < 0.05$ eV/atom)：** 仅保留落在凸包上或极其接近凸包的相，确保筛选出的结构在实验室中极大概率能被稳定合成。
4. **晶胞规模与有序度：** 限制单胞原子数在 20-60 之间，既能构成复杂的离子传导通道，又能避免 3D 点云处理时导致 GPU 显存溢出 (OOM)。同时，**严格剔除了带有分数占位 (Fractional Occupancy) 的无序结构**，确保符合 CDVAE 的输入标准。
