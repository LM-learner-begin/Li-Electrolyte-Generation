# CGCNN for Solid-State Electrolyte ECW Prediction
**基于晶体图卷积神经网络的电化学窗口预测**

本目录包含了大创项目中用于预测无机锂离子固态电解质电化学窗口 (Electrochemical Window, ECW) 的核心训练代码与参数配置。
本项目基于开源的 **CGCNN (Crystal Graph Convolutional Neural Networks)** 架构 https://github.com/txie-93/cgcnn
通过构建专用的电化学窗口数据集，并对模型超参数进行深度的系统性探索，成功建立了一套较高精度的预测模型。

---

## 数据集构建 (Dataset Construction)
为了让 CGCNN 能够学习到决定电化学窗口的三维结构特征，我们构建了专属的训练数据集：
* **数据来源：** 依托 Materials Project (MP) 数据库，使用热力学巨势相图 (GPPD) 方法计算得出的含锂无机化合物电化学窗口数据。
* **原始数据规模：** 16,001 条含锂化合物数据。
* **数据清洗：** 过滤掉无法构建正常晶体图邻接矩阵的气相分子与非致密结构，最终得到 **15,925** 条有效晶体数据。
* *(注：清洗后的专属数据集结构已上传至主仓库的 `dataset/` 目录。)*

---

## 模型调优过程 (Hyperparameter Optimization)

原始的 CGCNN 模型在预测复杂的电化学窗口时，出现了保守预测（低估高压材料）与验证集振荡（严重过拟合）的问题。本项目未更改底层网络源码，而是重点通过**模型超参数扩容**与**训练动力学调整**来攻克这一难题。

### Phase 1: 基础小模型测试
* **参数配置 (`64/3/128/1`):** 原子特征 64，卷积层 3，晶体特征 128，全连接层 1。Batch Size 256，初始学习率 0.01。
* **问题暴露：**
  * 30 轮训练后，模型对大于 5V 的高压材料预测极其保守（$R^2 = 0.619$）。
  * 延长至 500 轮后，训练集与验证集的 MAE 曲线出现巨大分歧，过拟合程度高达 41%。

### Phase 2: 大模型架构扩容 (`128/4/256/2`)
为提升模型对复杂多阴离子骨架的非线性拟合能力，我们全面放大了模型参数：
* **`--atom-fea-len 128`:** 提升原子特征维度，捕捉更细腻的电负性与半径差异。
* **`--n-conv 4`:** 增加卷积搜索深度，扩大图感受野。
* **`--h-fea-len 256`:** 提升晶体全局特征维度。
* **`--n-h 2`:** 增加输出端隐藏层数量，强化回归能力。

### Phase 3: 缓解过拟合的训练策略优化
在使用大模型后，为了压制过拟合趋势，我们进行了精细的动力学控制：
* **动态学习率衰减 (LR Schedule):** 采用分段衰减：Epoch 0-100 `0.005`，100-150 `0.0005`，150-200 `0.00005`，保障稳定收敛。
* **缩小批次大小 (`--batch-size 64`):** 将 Batch Size 降至 64，增加参数更新频次，利用小批次梯度噪声帮助模型跳出局部最优。
* **引入 Dropout (`p=0.2`):** 在全连接层前引入随机失活机制，强制阻断过度依赖的特征路径。

---

## 最佳模型表现 (Best Performance)
通过上述调优，在 **Epoch=200, Batch Size=64, 大模型参数并结合 Dropout** 的配置下，我们得到了最佳的电化学窗口预测模型：
* **测试集 MAE:** **0.181 V**。
* **过拟合抑制：** 训练集与验证集的差距缩小至 0.055，过拟合现象得到显著控制。
* *(注：最佳权重文件已保存结果文件的 `epoch200_upgrade_batch/` 目录。)*

## 运行指令参考 (How to Run)

可以直接使用我们探索出的最优参数组合，启动模型训练：
```bash
python main.py data/stability-all/ \
  --train-ratio 0.8 --val-ratio 0.1 --test-ratio 0.1 \
  --epochs 200 --batch-size 64 --workers 8 \
  --atom-fea-len 128 --n-conv 4 --h-fea-len 256 --n-h 2 \
  --optim Adam > train_log.txt 2>&1
